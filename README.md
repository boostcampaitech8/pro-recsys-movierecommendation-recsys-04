# 영화 추천 시스템 (Movie Recommendation System)

## 1. 프로젝트 개요

이 프로젝트는 다양한 추천 알고리즘을 구현하고 성능을 비교 분석하기 위한 영화 추천 시스템입니다. 최신 딥러닝 기반의 순차 추천 모델부터 전통적인 협업 필터링 모델에 이르기까지 폭넓은 스펙트럼의 추천 모델을 다룹니다. PyTorch Lightning을 활용하여 모델의 학습 및 평가 파이프라인을 체계적으로 구축하였으며, Optuna를 이용한 하이퍼파라미터 튜닝, Streamlit을 활용한 데이터 탐색 도구 등 연구 개발의 효율성을 높이기 위한 다양한 도구를 포함하고 있습니다.

## 2. 주요 기능

- **다양한 추천 모델 구현**: 여러 유형의 추천 모델(순차, 협업 필터링, 앙상블 등)을 구현하여 성능을 비교할 수 있습니다.
- **체계적인 학습/평가 파이프라인**: PyTorch Lightning을 기반으로 재사용 가능하고 모듈화된 학습 코드를 구성했습니다.
- **하이퍼파라미터 최적화**: Optuna를 사용하여 주요 모델의 하이퍼파라미터를 효율적으로 탐색하고 최적화합니다.
- **데이터 탐색 도구**: Streamlit으로 제작된 인터랙티브 대시보드를 통해 사용자의 평가 기록, 아이템 정보, 데이터 통계 등을 시각적으로 분석할 수 있습니다.
- **앙상블 기법 연구**: 여러 모델의 예측 결과를 결합하여 추천 성능을 향상시키는 앙상블 방법을 실험합니다.

## 3. 구현된 추천 모델

이 프로젝트에서는 다음과 같은 다양한 추천 모델을 구현하고 실험합니다.

- **순차(Sequential) 모델**:
  - `BERT4Rec`: Self-Attention을 활용하여 사용자의 행동 순서를 모델링하는 모델
  - `S3Rec`: Self-Attention 기반 모델에 추가적인 Self-Supervised Learning 목표를 더해 데이터 희소성 문제를 완화한 모델

- **정적(Static) / 협업 필터링 모델**:
  - `EASE`: 아이템 기반 협업 필터링을 단순화한 모델로, 학습 과정 없이 좋은 성능을 보입니다.
  - `SLIM` / `ADMMSLIM`: 희소 선형 모델(Sparse Linear Model)을 사용하여 아이템 간의 관계를 학습합니다.
  - `Multi-VAE` / `Rec-VAE`: Variational Autoencoder (VAE)를 활용하여 사용자-아이템 행렬의 잠재 표현을 학습합니다.
  - `LightGCN`: 그래프 신경망(GCN)을 단순화하여 사용자-아이템 상호작용 그래프를 효과적으로 학습합니다.
  - `FM (Factorization Machines)`: 고차원 희소 데이터에서 특징 간의 상호작용을 효과적으로 모델링합니다.

- **기타 모델**:
  - `CatBoost`: Gradient Boosting 기반의 모델로, 다양한 정형 데이터를 활용한 추천에 사용될 수 있습니다.
  - `Autoencoder`: 기본적인 오토인코더 구조를 사용한 추천 모델입니다.

## 4. 기술 스택

- **언어**: `Python 3.x`
- **머신러닝/딥러닝**: `PyTorch`, `PyTorch Lightning`, `Scikit-learn`
- **추천 라이브러리**: `RecBole`
- **하이퍼파라미터 튜닝**: `Optuna`
- **데이터 분석**: `Pandas`, `NumPy`
- **데이터 탐색 앱**: `Streamlit`
- **설정 관리**: `YAML`, `Hydra`

## 5. 디렉토리 구조

```
.
├── data_explorer/    # Streamlit 기반 데이터 탐색 및 시각화 도구
├── eda/              # 초기 데이터 분석(EDA) 관련 노트북 및 스크립트
├── ensemble/         # 여러 모델의 예측을 결합하는 앙상블 모델
├── lightning/        # PyTorch Lightning 기반 모델 구현 및 학습/추론 파이프라인
│   ├── configs/      # 모델 및 학습 과정 설정 (YAML)
│   ├── src/          # 모델, 데이터로더, 유틸리티 등 핵심 소스 코드
│   ├── tests/        # 단위/통합 테스트 코드
│   └── tune/         # Optuna를 사용한 하이퍼파라미터 튜닝
├── recbole/          # RecBole 라이브러리를 사용한 실험 코드
└── src/              # (Lightning 이전) 다양한 추천 모델 개별 구현체
```
